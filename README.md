## How to Run

1. **Install dependencies**

    ```bash
    npm install
    ```

2. **Create an `.env` file**

    ```bash
    touch .env
    ```

    You can find the keys needed in the `example.env` file.

3. **Run in development mode**

    ```bash
    npm run dev
    ```

4. **Run in production mode**

    ```bash
    npm start
    ```
